## What Goes Here?

Post in **this repo** for:
- Cross-project feature ideas and proposals
- Architectural decisions affecting multiple projects
- Questions like "Which Biocommons tool should I use for...?"
- Strategy and roadmap discussions
- General Biocommons questions

Post in **project repos** for:
- Bugs in a specific tool
- Feature requests for a single project
- Project-specific usage questions
- Implementation details

**Not sure?** Start here and we'll help you find the right place!
